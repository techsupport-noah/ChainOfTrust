## INSTALLATION

- Clone this repo.
- Unzip, if needed, and open the folder in an editor of your choice, for example use VS Code.
- From the terminal run: 
```
 cd Front_End
 npm install
```

## Execution

- From the terminal run: 
```
 npm run dev
```

## Export Commands

Generate: 
```
$ npm run generate
```
- Generates a static website ready for deployment.
